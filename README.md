# Mental Health Risk Signal Detector

A Streamlit-based research prototype for detecting mental health risk signals in Reddit-style text using TF-IDF and Logistic Regression.

## Features
- Single-text analysis
- Batch CSV upload
- Balanced Mode and Safety Mode
- Risk score, uncertainty, and review flag
- Downloadable prediction log
- Downloadable batch results

## Important Note
This is a research prototype for triage support. It is **not** a clinical diagnosis tool.
